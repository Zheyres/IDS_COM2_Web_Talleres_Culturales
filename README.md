Introduccion.
Contexto del Proyecto:
Esta aplicación web tiene como objetivo facilitar la difusión de talleres recreativos vinculados al centro cultural de la zona. La plataforma permite la inscripción online para los usuarios interesados y el registro de nuevos talleres por parte de colaboradores, previa moderación del contenido.

Objetivo de negocio:
El propósito es vincular a los residentes de la zona con la oferta educativa y cultural local, permitiendo que los usuarios se informen e inscriban de forma digital, y brindando a los talleristas un espacio formal para publicar y promocionar sus actividades.

Objetivo del Sistema:
Proveer una herramienta interactiva que centralice la información de los talleres. El sistema contará con un buscador y un mapa (utilizando la librería Leaflet) para localizar geográficamente las sedes. Ofrecerá los detalles de contacto y la información pertinente de cada taller, así como un formulario de inscripción en línea.

Asimismo, el sistema incluirá un módulo de administración para que el Centro Cultural realice la moderación de talleres y usuarios mediante herramientas de aprobación, rechazo y suspensión. Adicionalmente, contará con una funcionalidad de exportación de datos para generar una revista informativa mensual mediante un servicio externo.

Objetivo del Documento:
Este documento tiene como fin detallar los requisitos, especificaciones técnicas y alcances necesarios para la realización del proyecto, sirviendo como guía para su desarrollo y posterior validación.
